A small library that helps integrate smartcard-based authentication into web apps and services.

## Highlights

- **Secure**: hardware-backed keys
- **Portable**: works across major browsers and platforms

## Tech

Built with Node.js, WebAuthn, and custom middleware.

```ts
// Example usage
import { verify } from "@chysomm/smartcard";

verify(request);
```
