JufoPay is a lightweight payments dashboard and API to handle subscriptions and invoicing for small merchants.

## Features

- Recurring billing
- Webhooks and notifications
- Simple dashboard for managing customers

## Built with

Next.js, Stripe, and a serverless backend.

```js
// Initialize
const client = new JufoPay({ apiKey: process.env.JUFO_KEY });
await client.createInvoice({ amount: 2500, currency: "USD" });
```
